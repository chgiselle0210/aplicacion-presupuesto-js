# Proyecto Final del curso de Programación con JavaScript I

## Descripción

Este proyecto corresponde al entregable final del curso **Programación con JavaScript I**.  

Se desarrolló una aplicación Web interactiva para la administración de presupuesto, que hace posible registrar ingresos y egresos, calcular totales dinámicamente y observar el porcentaje que representan los egresos respecto a los ingresos.

Con este proyecto, se ponen en práctica los conocimientos adquiridos a lo largo de los temas y avances semanales del curso.

---

## Objetivo del proyecto

Aplicar los conocimientos aprendidos en JavaScript para desarrollar una página web interactiva que:

- Manipule el DOM dinámicamente.
- Utilice clases y herencia.
- Realice cálculos en tiempo real.
- Dar lugar a la interacción del usuario mediante formularios.
- Muestre información formateada correctamente.

---

## Funcionalidades ejecutadas

- Registro dinámico de ingresos y egresos.
- Cálculo automático del presupuesto disponible.
- Cálculo del porcentaje de egresos respecto a ingresos.
- Formateo de números como moneda (MXN).
- Actualización dinámica del HTML mediante `innerHTML`.
- Eliminación de registros individuales.
- Uso de clases (`Dato`, `Ingreso`, `Egreso`) con herencia.
- Manejo de arreglos para almacenamiento de datos.
- Validación básica de formulario.
